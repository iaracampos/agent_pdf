# Sistema Inteligente de Perguntas e Respostas para Documentos PDF

Um sistema de Recuperação Aumentada por Geração (RAG) em Python que permite fazer perguntas em linguagem natural sobre documentos PDF e obter respostas precisas e contextualizadas.

## Funcionalidades
- Upload múltiplo de PDFs
- Processamento em chunks otimizados
- Busca semântica com embeddings vetoriais
- Geração de respostas com LLM
- Avaliação automática da qualidade das respostas

## Tecnologias Utilizadas
- LangChain
- FAISS
- Hugging Face Embeddings (all-mpnet-base-v2)
- Mistral AI
- RAGAS
- PyPDF

## Pré-requisitos
- Python 3.8+
- Google Colab
- Chave de API do Mistral AI (deve ser gerada no site mistral )

## Documentos Utilizados no RAG
